使用npm管理其他的包

2019-01-09

1.创建自己的包

创建my-project 目录
mkdir my-project
切换到my-project目录下
cd my-project
创建 package.json文件
npm init

2.安装第三方的包

全局安装
全局安装的包可以在任何目录下使用，一般情况下，提供命令行工具的包推荐全局安装。

本地安装
本地安装的包只能在安装目录下使用。

开发阶段

如果我们只是在开发阶段依赖某个包，则在安装这包时，可以跟上 –save-dev或 -D 选项。

npm install --save-dev gulp

产品阶段

如果我们在开发阶段和产品阶段都依赖某个包，则在安装这包时，可以跟上 –save 选项。

npm install --save jquery

删除已安装的包

npm uninstall --save jQuery

更新已安装的包

npm update --save jquery

列出已安装的包

npm list

解决npm下载慢的问题

方法一：使用淘宝定制的 cnpm 命令行工具代替默认的 npm

npm install -g cnpm --registry=https://registry.npm.taobao.org

方法二：修改npm的下载仓库为淘宝镜像

npm config set registry http://registry.npm.taobao.org/

如果要发布自己的镜像需要修改回来

npm config set registry https://registry.npmjs.org/
